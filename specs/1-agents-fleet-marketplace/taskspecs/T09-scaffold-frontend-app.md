# T09 — Scaffold Frontend App

Purpose
- Create a minimal Next.js + Tailwind starter configured for SSR/ISR, i18n (AR/EN), and theme switching.

Scope
- Repo scaffold, routing conventions, global layout, header/footer, env examples, and basic ESLint/Prettier config.

Deliverables
- Starter app in `apps/frontend/` with npm scripts: dev, build, start, lint, test.

Acceptance Criteria
- `npm run dev` boots a working site showing a placeholder product grid and theme switcher.

Estimated time: 1–2 days
Owner: Frontend Engineer
Dependencies: T01, T06
