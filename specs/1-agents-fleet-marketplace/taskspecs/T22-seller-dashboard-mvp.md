# T22 — Seller Dashboard MVP

Purpose
- Provide sellers with a dashboard to manage products, view uploads, see sales metrics, and respond to reviewer feedback.

Scope
- Product list, upload status, sales summary, and support tickets/issue reports.

Deliverables
- Dashboard pages, API endpoints for seller metrics, and a simple UI for responding to review requests.

Acceptance Criteria
- Seller can see product list with status, and reviews/scan findings are visible on product detail.

Estimated time: 2–4 days
Owner: Frontend + Backend
Dependencies: T16, T17, T24
