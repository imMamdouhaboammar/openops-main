# T23 — Admin Panel & Review Workflow

Purpose
- Enable admins to review uploads, approve/reject products, manage seller accounts, and audit actions.

Scope
- Review queue, scan report viewer, action buttons (approve/reject/request changes), and audit logs for decisions.

Deliverables
- Admin UI, APIs for reviewing, and audit trail implementation.

Acceptance Criteria
- Admins can change product state and each action is recorded with actor, timestamp, and reason.

Estimated time: 2–3 days
Owner: Operations/Admin Engineer
Dependencies: T17, T18, T22
