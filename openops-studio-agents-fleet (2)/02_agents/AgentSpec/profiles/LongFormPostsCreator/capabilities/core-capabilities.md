## 3. Long-Form Structure (2000+ words)

### Hero's Journey

```
Hook → Background → Challenge → Attempts → Turning Point → Solution → CTA
```

**Example (Egyptian)**:

```
[Hook] في 2019، خسرت 300 ألف جنيه في 6 شهور.

[Background] أنا ممدوح، مهندس برمجيات، كان عندي حلم...

[Challenge] المشكلة إن أنا عملت منتج 9 شهور بدون عملاء...

[Attempts] حاولت Facebook Ads: 5000 جنيه، 0 customers
حاولت LinkedIn Outreach: 500 messages, 3 replies

[Turning Point] لحد ما mentor قالي: "أنت بتبيع solution، مش product"

[Solution] كلمت 50 عميل potential، فهمت مشاكلهم، غيّرت positioning...

[CTA] لو أنت بتفكر تعمل مشروع، اتكلم مع 50 عميل الأول.
```

### Framework Post

```
Problem → Why It Matters → Framework (3-7 steps) → Deep Dive → Case Study → CTA
```

**Example (Saudi)**:

```
[Problem] الحين كثير ناس يبون يبدون مشاريع، بس ما يعرفون من وين...

[Framework] الإطار الخماسي:
1. تحديد المشكلة
2. بناء MVP
3. التسعير المبكر
4. أول 10 عملاء
5. التحسين المستمر

[Case Study] أحمد من الرياض: Week 1 → 25 عميل potential...

[CTA] مين يبي يبدأ؟ ابدأ بالخطوة الأولى: كلّم 20 شخص هالأسبوع.
```

## 4. Platform Optimization

### LinkedIn

- **Length**: 1300-2000 words
- **Hook**: 3 lines, 70 chars max per line
- **Paragraphs**: 1-2 sentences
- **Emojis**: 3-7 total
- **Hashtags**: 3-5 (Arabic + English)

**Hook Formula**:

```
❌ Weak: "النهاردة هتكلم عن التعلم المستمر"
✅ Strong: "خسرت 200 ألف جنيه عشان ماكنتش بتعلم الحاجة دي"
```

---

### Twitter Threads

- **Length**: 10-15 tweets
- **Structure**:
  - Tweet 1: Hook + "thread 🧵"
  - Tweets 2-9: Content with examples
  - Tweet 10: Summary
  - Last tweet: CTA + engagement ask

**Example Start (Egyptian)**:

```
1/12 🧵 خسرت 200 ألف جنيه في 6 شهور.

السبب: غلطة واحدة معظم الناس بتقع فيها.

هقولك الغلطة + الحل في الـ thread ده:

---

2/12 الغلطة: إنك تبني منتج بدون ما تكلم عملاء.

أنا عملت منتج 9 شهور، كنت فاكر الناس هتشتريه لأنه "حلو".

الحقيقة: مفيش حد يشتري حاجة "حلوة"، الناس بتشتري حلول.
```
