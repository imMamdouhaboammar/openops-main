
## 📝 PROMPT ENGINEERING RULES

### ✅ DO (Narrative Style)

```
"A sleek chrome robot barista with articulated fingers carefully 
operates an espresso machine, steam rising in the low-gravity 
environment of a futuristic cafe on Mars. Through the dome windows, 
the rusty Martian landscape stretches to the horizon under a pink sky."
```

### ❌ DON'T (Keyword Style)

```
"robot, coffee, futuristic, Mars, cafe, chrome, 3D"
```

### Prompt Structure

```
1. Subject Description (Who/What - detailed appearance)
2. Action (What is happening)
3. Environment (Where - specific details)
4. Technical (Camera angle, lighting, composition)
5. Style (Art style, mood, reference)
6. Specifications (Resolution, aspect ratio)
```

---

## ⚖️ QUALITY ASSESSMENT CRITERIA

| Criterion | Weight | What to Check |
|-----------|--------|---------------|
| **Subject Accuracy** | 25% | Does subject match description? |
| **Composition Match** | 20% | Does layout match blueprint? |
| **Style Fidelity** | 20% | Does style match specification? |
| **Technical Quality** | 15% | Resolution, sharpness, artifacts? |
| **Text Accuracy** | 10% | If text, is it correct? |
| **Mood/Atmosphere** | 10% | Does emotional tone match? |

### Thresholds

- **≥ 0.85**: ✅ PASS → Output
- **≥ 0.70**: ⚠️ CONDITIONAL → Minor refinement
- **< 0.70**: ❌ FAIL → Return to Stage 4

---

## 🔄 REFINEMENT STRATEGIES

| Issue | Strategy |
|-------|----------|
| **Subject wrong** | Regenerate with clearer subject prompt |
| **Composition off** | Add explicit composition guidance |
| **Style mismatch** | Add style reference images |
| **Technical issues** | Adjust resolution, add quality keywords |
| **Text errors** | Regenerate with quoted, explicit text |
| **Mood wrong** | Adjust lighting and color descriptions |

**Max Iterations:** 3
