# Core Capabilities - Startup Growth Strategist

## 1. UpgradIQ Startup Table Framework

### Overview

The **UpgradIQ Startup Table** is the foundational operating map that structures all startup growth strategies. It's a comprehensive, color-coded framework covering **7 stages** with **100+ elements** that answer:

> "إيه الصح اللي محتاج يتعمل دلوقتي؟ وإيه اللي بعده؟ وليه؟"

### Framework Philosophy

**Not a theory. An operating map.**

- **Sequenced**: Every element has prerequisites and next steps
- **Stage-Appropriate**: What works at Seed kills at Pre-Seed
- **Comprehensive**: Strategy + Product + Growth + Ops + Finance
- **Actionable**: Designed for execution, not contemplation

## 2. Stage Diagnosis & Assessment

### Capability Overview

Accurately identify where a startup is in its lifecycle and assess readiness for progression.

### Diagnostic Process

#### Step 1: **Metrics Collection**

```
Financial:
- MRR/ARR
- Burn rate
- Runway
- Gross margin

Growth:
- CAC
- LTV
- Payback period
- Churn rate

Product:
- DAU/MAU
- Activation rate
- Feature adoption
- NPS

Team:
- Team size
- Key roles filled
- Advisor/board composition
```

#### Step 2: **Benchmark Comparison**

Compare collected metrics against stage-specific benchmarks:

| Metric | Pre-Seed | Seed | Series A | Series B |
|--------|----------|------|----------|----------|
| MRR | <$10K | $10K-$50K | $100K-$250K | $1M+ |
| Team | 1-5 | 5-15 | 15-50 | 50-150+ |
| Customers | 10-50 | 50-500 | 500-2000 | 2000+ |
| CAC Payback | N/A | <18mo | <12mo | <12mo |

#### Step 3: **Capability Audit**

Assess which UpgradIQ Table elements are in place:

**Foundation Stage Elements** (Blue):

- ✅ PRC (Problem-Solution Fit): Validated
- ✅ CSD (Customer Discovery): 50+ interviews
- ⚠️ MKS (Market Sizing): Rough estimate only
- ❌ VLP (Value Proposition): Not clearly defined

**Scoring**:

- ✅ = 100% (2 points)
- ⚠️ = 50% (1 point)
- ❌ = 0% (0 points)

**Total Score**: 5/8 = 62.5% stage completion

#### Step 4: **Gap Analysis**

Identify critical missing elements:

**Critical Gaps** (blocking next stage):

1. Value Proposition clarity
2. MVP not launched
3. No paying customers

**Secondary Gaps** (should address soon):

1. Market sizing incomplete
2. No onboarding system
3. Metrics tracking ad-hoc

#### Step 5: **Readiness Score**

Calculate readiness for next stage (0-100):

```
Readiness Score = 
  (Metrics Score × 40%) + 
  (Capability Completeness × 40%) + 
  (Team Fit × 20%)

Example:
- Metrics Score: 60/100
- Capability: 62.5/100
- Team Fit: 75/100

Readiness = (60×0.4) + (62.5×0.4) + (75×0.2) = 64/100
```

**Interpretation**:

- **0-40**: Not ready, significant work needed
- **41-60**: Early prep, 6+ months away
- **61-75**: Preparation phase, 3-6 months
- **76-85**: Nearly ready, 1-3 months
- **86-100**: Ready to progress

### Output: Stage Assessment Report

```markdown
## Startup: [Name]
## Current Stage: Pre-Seed
## Readiness for Seed: 64/100 (Preparation Phase)

### Strengths
- Strong problem validation (50+ interviews)
- Experienced founding team
- Clear target market

### Critical Gaps (Must Fix)
1. Value proposition needs sharpening
2. MVP completion (60% done)
3. No paying customers yet

### Recommended Timeline
- 3-6 months to Seed readiness
- Focus: Launch MVP, get 10 paying customers

### Next 90 Days Priority
[See Playbook Generation]
```

## 3. Priority Matrix Generation

### Capability Overview

Generate stage-specific priority matrix using adapted Eisenhower framework.

### Framework: Startup Priority Matrix

|  | **Urgent** | **Not Urgent** |
|---|-----------|---------------|
| **Important (Stage-Critical)** | **DO NOW** (This week/month) | **PLAN** (This quarter) |
| **Not Important (Nice-to-Have)** | **DELEGATE** | **ELIMINATE** |

### Stage-Specific Matrix Examples

#### Pre-Seed Example

| **DO NOW** | **PLAN** |
|-----------|---------|
| - Complete customer interviews (20 more) | - Build MVP roadmap |
| - Define value proposition | - Identify potential advisors |
| - Validate pricing hypothesis | - Research funding sources |

| **DELEGATE** | **ELIMINATE** |
|-------------|-------------|
| - Social media management | - Building perfect pitch deck |
| - Basic bookkeeping | - Networking events (low ROI) |
| | - Building advanced features |

#### Seed Example

| **DO NOW** | **PLAN** |
|-----------|---------|
| - Fix activation rate (currently 20%) | - Hire first sales hire |
| - Build retention systems | - Establish board of advisors |
| - Implement analytics properly | - Series A fundraising prep |
| - Get CAC payback <18 months | - Build referra program |

| **DELEGATE** | **ELIMINATE** |
|-------------|-------------|
| - Customer support (Tier 1) | - Building enterprise features |
| - Content creation | - Multi-region expansion |
| - Basic operations | - Complex automation |

### Prioritization Rules by Stage

**Pre-Seed**:

- **Do**: Validate, validate, validate
- **Don't**: Build, scale, hire prematurely

**Seed**:

- **Do**: Build repeatable systems, find PMF signals
- **Don't**: Premature scaling, ignore unit economics

**Series A**:

- **Do**: Scale what works, systematize operations
- **Don't**: Chase new markets, lose unit economics discipline

**Series B+**:

- **Do**: Build enterprise infrastructure, defend moat
- **Don't**: Lose focus, ignore profitability path

## 5. Investor Readiness Assessment

### Capability Overview

Evaluate fundraising preparation across three dimensions: Metrics, Story, Process.

### Assessment Framework

#### Dimension 1: Metrics Readiness (40% weight)

**Scoring Rubric (1-10 per metric, average)**:

| Score | MRR/ARR vs. Stage | Growth Rate | Unit Economics | Retention |
|-------|-------------------|-------------|----------------|-----------|
| 10 | Top 10% percentile | >20% MoM | LTV/CAC >4x, Payback <6mo | Cohort retention >60% |
| 7-9 | Top 25% | 10-20% MoM | LTV/CAC >3x, Payback <12mo | Retention >40% |
| 4-6 | Median | 5-10% MoM | LTV/CAC >2x, Payback <18mo | Retention >25% |
| 1-3 | Below median | <5% MoM | LTV/CAC <2x, Payback >18mo | Retention <25% |

**Example**:

- MRR vs. Stage: 8/10 (top 20%)
- Growth: 6/10 (8% MoM)
- Unit Economics: 7/10 (LTV/CAC = 3.2x, Payback 11mo)
- Retention: 5/10 (32% Month 3)

**Metrics Score**: (8+6+7+5)/4 = **6.5/10**

---

#### Dimension 2: Story Readiness (30% weight)

**Elements** (Yes/No, convert to 1-10):

- [ ] **Problem Clarity**: Can explain problem in 30 seconds? (2 points)
- [ ] **Solution Uniqueness**: Clear differentiation? (2 points)
- [ ] **Market Sizing**: TAM/SAM/SOM calculated? (1 point)
- [ ] **Traction Narrative**: Impressive growth story? (2 points)
- [ ] **Team Completeness**: Key roles filled? (1 point)
- [ ] **Vision**: Compelling 5-year vision? (1 point)
- [ ] **Competition**: Honest competitive analysis? (1 point)

**Score**: X/10 points → X/10

**Example**: 7/10 points = **7/10**

---

#### Dimension 3: Process Readiness (30% weight)

**Elements**:

| Element | Status | Score |
|---------|--------|-------|
| Financial Model (3-year) | ✅ Complete | 2/2 |
| Data Room | ⚠️ 60% ready | 1/2 |
| Cap Table (clean) | ✅ Clean | 2/2 |
| Legal (incorporation, IP) | ✅ Good | 2/2 |
| Pitch Deck | ⚠️ Needs work | 1/2 |
| Management Presentations | ❌ Not ready | 0/2 |

**Score**: 8/12 points → **6.7/10**

---

#### Overall Investor Readiness Score

```
Final Score = 
  (Metrics × 0.4) + (Story × 0.3) + (Process × 0.3)

Example:
  (6.5 × 0.4) + (7.0 × 0.3) + (6.7 × 0.3)
  = 2.6 + 2.1 + 2.01
  = 6.71/10
```

**Interpretation**:

- **8-10**: Highly ready, start outreach now
- **6.5-7.9**: Ready with minor improvements (1-2 months)
- **5-6.4**: Need 3-6 months preparation
- **<5**: Significant gaps, 6+ months

### Output: Investor Readiness Report

```markdown

## Startup: [Name]
## Date: [Date]

Week 1-2: [Actions]
```
