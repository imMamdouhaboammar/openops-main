# Content Repurposing Framework

## 🎯 Purpose

Transform one piece of content into multiple formats optimized for different platforms, maximizing reach and efficiency while maintaining message consistency.

---

## 📊 Content Repurposing Matrix

### From Long-Form Blog Post (1500+ words)

**To Instagram:**

- 📸 **Carousel (10 slides)**: Key points + visuals
- 🎬 **Reel (30-60s)**: Quick summary with hook
- 📝 **Caption**: Main takeaway + CTA
- 📊 **Infographic**: Data visualization

**To Twitter/X:**

- 🧵 **Thread (8-12 tweets)**: Breakdown of main points
- 📊 **Quote Cards**: Highlight key insights
- 🔗 **Link Post**: Teaser + link

**To TikTok:**

- 🎥 **Educational Video (60s)**: Quick tips format
- 🎭 **Trend Integration**: Main point in trending format

**To LinkedIn:**

- 📄 **Article**: Professional version
- 📊 **Document Post**: Carousel-style
- 💬 **Discussion Post**: Thought-provoking question

**To YouTube:**

- 🎬 **Short (60s)**: Quick summary
- 📺 **Full Video (5-10 min)**: Detailed explanation

---

### From Instagram Reel (30-60s)

**To TikTok:**

- ✂️ **Direct Repost**: Adjust format (9:16)
- 🎵 **Trend Adaptation**: Use trending sound

**To YouTube Shorts:**

- 📱 **Direct Upload**: Same content
- 📝 **Add Chapters**: For longer versions

**To Twitter/X:**

- 🎬 **Native Video**: Upload directly
- 🖼️ **GIF**: Key moment extraction

**To LinkedIn:**

- 🎥 **Professional Version**: Add context
- 💼 **Behind-the-Scenes**: Process explanation

**To Blog:**

- 📝 **Transcript + Expansion**: Full article
- 🖼️ **Screenshots**: Visual breakdown

---

## 🔄 Repurposing Workflow

### Step 1: Content Audit

**Identify source content:**

- ✅ High-performing content
- ✅ Evergreen topics
- ✅ Comprehensive pieces
- ✅ Unique insights

### Step 2: Platform Selection

**Choose target platforms based on:**

- 🎯 Audience presence
- 📊 Platform strengths
- ⏰ Content lifespan
- 🎨 Format compatibility

### Step 3: Format Adaptation

**For each platform:**

1. **Extract core message**
2. **Adapt to platform norms**
3. **Optimize for algorithm**
4. **Maintain brand voice**

### Step 4: Content Atomization

**Break down into:**

- 🎯 **Hooks**: Attention-grabbers
- 💡 **Key Points**: Main insights
- 📊 **Data/Stats**: Supporting evidence
- 🖼️ **Visuals**: Graphics/images
- 💬 **Quotes**: Memorable lines
- ✅ **CTAs**: Action items

### Step 5: Optimization

**Platform-specific:**

- 📝 **Copy length**: Platform limits
- 🎨 **Visual format**: Aspect ratios
- #️⃣ **Hashtags**: Platform strategy
- ⏰ **Timing**: Optimal posting
- 🎵 **Audio**: Music/voiceover

### Step 6: Batch Creation

**Efficiency tips:**

- 📅 Create all versions at once
- 🎨 Use templates
- ♻️ Reuse assets
- 🤖 Automate where possible

## 🎨 Platform-Specific Adaptation Rules

### Instagram

- **Visual-first**: High-quality images/videos
- **Carousel**: 7-10 slides optimal
- **Reels**: 7-15s hook, 30-60s total
- **Captions**: 125-150 characters for preview
- **Hashtags**: 3-5 relevant

### TikTok

- **Hook**: First 3 seconds critical
- **Trend**: Leverage trending sounds
- **Text**: On-screen text for clarity
- **Length**: 15-60s optimal
- **CTA**: Comment engagement

### Twitter/X

- **Thread**: 8-12 tweets optimal
- **Hook**: First tweet crucial
- **Visuals**: 1-4 images per tweet
- **Length**: 100-280 characters
- **Engagement**: Questions work well

### LinkedIn

- **Professional**: Business-focused
- **Length**: 1300-2000 characters
- **Format**: Line breaks for readability
- **Visuals**: Professional graphics
- **CTA**: Thought leadership

### YouTube

- **Thumbnail**: Eye-catching
- **Title**: Keyword-optimized
- **Description**: Detailed + timestamps
- **Shorts**: Vertical 9:16
- **Length**: Shorts 15-60s, Videos 7-15 min

## 🚫 Common Repurposing Mistakes

### ❌ Don't

1. **Copy-paste** without adaptation
2. **Ignore platform norms**
3. **Over-repurpose** (fatigue)
4. **Forget CTAs**
5. **Skip optimization**
6. **Neglect timing**
7. **Use wrong formats**
8. **Ignore analytics**

### ✅ Do

1. **Adapt** for each platform
2. **Respect** platform culture
3. **Space out** reposts (2-3 months)
4. **Include** clear CTAs
5. **Optimize** for algorithms
6. **Time** strategically
7. **Match** platform formats
8. **Track** performance

## 🎯 Best Practices

1. **Start with winners**: Repurpose top-performing content
2. **Batch create**: Do all versions at once
3. **Use templates**: Speed up process
4. **Maintain quality**: Don't sacrifice for quantity
5. **Test formats**: See what works per platform
6. **Update regularly**: Refresh old content
7. **Track results**: Learn and improve
8. **Automate**: Use tools where possible
