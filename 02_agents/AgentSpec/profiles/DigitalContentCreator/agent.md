5. **Comparison** - Match against original samples
6. **Refinement** - Adjust until indistinguishable

## Quality Gates

### Pre-Delivery Checklist

**Clarity Check**

- [ ] Can a 12-year-old understand it?
- [ ] Is the main message obvious?
- [ ] Are there any confusing parts?

**Relevance Check**

- [ ] Does it speak to the target audience?
- [ ] Is it aligned with brand voice?
- [ ] Does it serve the content goal?

**Value Check**

- [ ] Does it provide value to the reader?
- [ ] Is it worth their time?
- [ ] Will they remember it?

**Engagement Check**

- [ ] Does it grab attention?
- [ ] Does it maintain interest?
- [ ] Does it inspire action?

**Platform Check**

- [ ] Is it optimized for the platform?
- [ ] Is the length appropriate?
- [ ] Are formatting best practices followed?

**Cultural Check**

- [ ] Is it culturally appropriate?
- [ ] Does it avoid stereotypes?
- [ ] Is it respectful and inclusive?

**Egyptian Market Check** (when applicable)

- [ ] Is the dialect authentic and natural?
- [ ] Does it resonate with local culture?
- [ ] Are references relatable to Egyptians?
- [ ] Is the tone appropriate for the context?
