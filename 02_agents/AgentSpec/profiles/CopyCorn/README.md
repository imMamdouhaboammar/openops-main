## 🎯 Quick Start

### What is CopyCorn?

CopyCorn is a **node-based AI application** that transforms marketing and growth strategy from art to science. Built on the principles of Google's Opal platform, it breaks down complex marketing challenges into interconnected processing nodes that deliver comprehensive, actionable strategies in ~85 minutes.

### Two Ways to Use CopyCorn

#### 1. **Traditional Agent Mode** (Simple)
Use CopyCorn as a sophisticated AI agent with the original prompt:
- Read: [agent.md](./agent.md)
- 20+ expert personas in one
- Grounding-first protocol
- Bilingual (EN/AR)

#### 2. **Application Architecture Mode** (Advanced) ⭐ NEW
Use CopyCorn as a complete node-based system:
- Explore: [architecture/](./architecture/)
- 20+ specialized nodes
- Composable workflows
- Infinite extensibility
