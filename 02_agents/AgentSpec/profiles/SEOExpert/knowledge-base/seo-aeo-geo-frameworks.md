## AEO Best Practices

### Featured Snippet Types

1. **Paragraph** (40-60 words): Direct answer to "What is..." questions
2. **List** (Numbered OR bulleted): How-to steps, items in a category
3. **Table**: Comparisons, pricing, data

### "People Also Ask" (PAA) Optimization

- Target 3-5 related questions per page
- Use question format in H2/H3 tags
- Provide concise answers (100-150 words)
- Implement FAQPage schema

## GEO Strategies (ChatGPT/Gemini/Perplexity)

### Citation Authority Tactics

1. **Original Research**: Publish studies, data, surveys
2. **Expert Authorship**: Real credentials, author bios (LinkedIn profiles)
3. **Third-Party Mentions**: Get cited by .edu, .gov, industry sites
4. **Content Freshness**: Update every 3-6 months

### Schema Markup Priorities

- **Article**: All blog posts, guides
- **FAQPage**: FAQ sections
- **HowTo**: Step-by-step tutorials
- **Event**: Hajj dates, Ramadan calendar
- **Organization**: Brand entity
