# Spec 04: Design Tokens

## Goal

Codify the brand's visual identity into actionable CSS variables.

## brand Colors

- Primary Navy: #000033
- Primary Blue: #0068ff
- Accent Purple: #9347ff
- Accent Gold: #f1ba39
- Accent Green: #00c377
- Surfaces: #ffffff, #eef6ff, #f5eeff, #fcfaeb, #e7f7f0

## Typography

- Arabic: Alexandria (Alexandria font-family)
- English: Montserrat (Montserrat font-family)

## Implementation

- All tokens must be isolated in a global CSS or theme file.
- No hardcoded hex values in component styles.
